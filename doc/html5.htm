Document Structure

Tags: article aside blockquote body br details dialog div footer h1-h6 head header hr html main nav p pre section summary template wbr

These tags are used to add structure to an HTML document.

The simpliest valid document is:

<!DOCTYPE html>
<html>
  <head>
  </head>

  <body>
  </body>

</html>

Machine-readable Content

Tags: abbr acronym address base basefont data !DOCTYPE meta time title

These tags are used to add machine-readable content to the document. Machine-readable content is data that is not rendered. It is intended to be read by browser, web search tools and so on.

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
