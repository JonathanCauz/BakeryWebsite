Create Table

Use the CREATE command to create a new table:

CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, ....);

Drop Table

Use the DROP command to remove/delete a table:

DROP TABLE table_name;

Alter Table

Use the ALTER command to alter/modify a tables definitiion.

One can add,remove or modify a column definition. One can also modify the table's constraints and keys.

Examples:

Add Column:

ALTER TABLE table_name ADD column_name datatype;

Remove Column:

ALTER TABLE table_name DROP COLUMN column_name;

Change the data type of a column in a table, use the following syntax:

ALTER TABLE table_name MODIFY COLUMN column_name datatype;

Display schema of table

Use the describe command to display the schema (column definitions and so on) of a table. For example:

describe numbers;

Insert

Use the INSERT command to add rows to a table.

The basic form of an insert command is: 

INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);

Update

Use the UPDATE command to change the columns of rows in the table.

The basic form of an update command is: 

UPDATE table_name SET column1 = value1, column2 = value2, ...  WHERE condition;

Delete

Use the DELETE command to remove rows from a table.

The basic form of an delete command is: 

delete from table_name where condition;

For example, 

delete from accounts where last_name='cauz';

Select

Use the SELECT command to read data from one or more tables.

The most basic form of an select command is: 

select col1, col2 from table_name

Databases

All tables are stored in Databases.

From an mysql shell session, get the list of databases accessible by server with the command: 

show DATABASES;

In order to access the tables contained in database, it should be 'in use'. To use the database called sys, 

use sys;

Alternatively, you can access tables from a database by qualifying the name of the table with the database name. For example, to access table numbers that is in database example, use: 

select * from example.numbers;

To get a list of the tables contained in a database, use: 

show TABLES;

To create a new database, use the CREATE DATABASE command. For example: 

create database MarieBakery;

To drop/delete/remove a database, use the command DROP DATABASE. For example:

DROP DATABASE example;
